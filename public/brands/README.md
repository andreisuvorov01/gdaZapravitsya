# Логотипы сетей АЗС

`BrandBadge` автоматически показывает официальный логотип, если в этой папке
лежит файл с именем `<slug>.svg` (прозрачный, без фона). Если файла нет —
рисуется запасной нейтральный бейдж с инициалами и фирменным цветом.

Нужны прозрачные SVG (предпочтительно) с именами строго по slug:

| Сеть | Файл |
| --- | --- |
| Лукойл | `lukoil.svg` |
| Роснефть (вкл. ТНК, Башнефть) | `rosneft.svg` |
| Газпромнефть | `gazpromneft.svg` |
| Газпром | `gazprom.svg` |
| Татнефть | `tatneft.svg` |
| Shell | `shell.svg` |
| BP | `bp.svg` |
| Teboil | `teboil.svg` |
| Нефтьмагистраль | `neftm.svg` |
| ОПТИ | `opti.svg` |
| EKA | `eka.svg` |
| ПТК | `ptk.svg` |
| Ирбис | `irbis.svg` |
| ННК | `neftmag.svg` |

Рекомендации:
- Формат — **SVG** с прозрачным фоном (вписывается в квадрат, поля задаёт компонент).
- Используйте только логотипы, на которые у вас есть права (товарные знаки
  принадлежат правообладателям; допускается номинативное использование для
  идентификации сети на карте).
